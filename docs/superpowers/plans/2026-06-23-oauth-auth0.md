# OAuth 2.1 — Auth0 + Kawakami MCP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Persistir carrinho entre conversas via Auth0 OAuth 2.1 — user_id estável do JWT substitui session_id aleatório.

**Architecture:** FastMCP (`mcp==1.28.0`) já suporta `token_verifier` nativo. Implementamos um `TokenVerifier` que valida JWTs do Auth0 contra JWKS. O `criar_sessao` extrai `sub` do token → user_id. Ferramentas de catálogo ficam `noauth`, carrinho/listas `oauth2`. O ChatGPT gerencia o fluxo OAuth (CIMD + PKCE) automaticamente.

**Tech Stack:** Python 3.13, `mcp==1.28.0`, Auth0 (tenant gratuito), SQLite

---

## Pré-requisitos: Configurar Auth0 (manual, ~10 min)

Antes de qualquer código, criar no [Auth0 Dashboard](https://manage.auth0.com):

1. **Criar tenant** (gratuito, 25k MAU)
2. **Criar API** em Applications → APIs:
   - Name: `Kawakami MCP`
   - Identifier (audience): `https://kawakami.axischat.com.br`
   - Signing Algorithm: `RS256`
3. **Configurar Application** (a Default App que Auth0 cria):
   - Application Type: `Regular Web Application`
   - Token Endpoint Authentication: `None` (public client, PKCE)
   - Grant Types: `Authorization Code`, `Refresh Token`
   - Anotar: `Domain` (ex: `kawakami.us.auth0.com`), `Client ID`, `Client Secret` (não usar)

```python
# src/infrastructure/auth0_verifier.py
import logging
from mcp.server.auth.provider import AccessToken, TokenVerifier
import jwt
from jwt import PyJWKClient

from src.config import settings

logger = logging.getLogger(__name__)


class Auth0TokenVerifier(TokenVerifier):
    """Valida JWTs do Auth0 usando JWKS."""

    def __init__(self) -> None:
        if not settings.auth0_domain:
            raise RuntimeError("KWK_AUTH0_DOMAIN is required for OAuth")
        self._jwks = PyJWKClient(settings.auth0_jwks_url)
        self._audience = settings.auth0_audience
        self._issuer = settings.auth0_issuer

    async def verify_token(self, token: str) -> AccessToken | None:
        try:
            signing_key = self._jwks.get_signing_key_from_jwt(token)
            claims = jwt.decode(
                token,
                signing_key.key,
                algorithms=["RS256"],
                audience=self._audience,
                issuer=self._issuer,
                options={"require": ["exp", "iss", "sub", "aud"]},
            )
        except jwt.ExpiredSignatureError:
            logger.warning("Token expired")
            return None
        except jwt.InvalidTokenError as e:
            logger.warning("Invalid token: %s", e)
            return None

        scopes_str = claims.get("scope", "")
        scopes = scopes_str.split() if scopes_str else []

        return AccessToken(
            token=token,
            client_id=claims.get("azp", ""),
            scopes=scopes,
            expires_at=claims.get("exp"),
            resource=claims.get("aud", ""),
        )
```

O FastMCP injeta o `AccessToken` no request context. Precisamos acessar via `ctx` ou middleware.

Verificar: `mcp.server.auth.middleware.auth_context` — pode conter `AccessToken`.

```python
# Em sessions.py:
from mcp.server.auth.middleware import auth_context
from mcp.server.fastmcp import Context

@safe_tool
async def criar_sessao(ctx: Context) -> CallToolResult:
    # Tentar pegar user_id do token OAuth
    token = auth_context.get()  # AccessToken ou None
    if token:
        # Extrair sub claim — precisamos decodificar o JWT
        import jwt
        claims = jwt.decode(token.token, options={"verify_signature": False})
        user_id = claims.get("sub", "")
        if user_id:
            # Criar/garantir sessão por user_id
            if not session_store.session_exists(user_id):
                session_store.create_session_with_id(user_id)
            return session_result(user_id)
    # Fallback: UUID aleatório (modo sem auth)
    return session_result(session_store.create_session())
```

# Em session_store.py:
def create_session_with_id(self, session_id: str) -> str:
    with self._get_conn() as conn:
        conn.execute(
            "INSERT OR IGNORE INTO sessions (id, created_at) VALUES (?, ?)",
            (session_id, time.time()),

    return session_id
